# Contexto de negocio

- Los datos representan un entorno retail sintético.
- Las tiendas están ubicadas en diferentes ciudades (ejemplo: Madrid, Barcelona, etc.).
- Las categorías principales incluyen: Bebidas, Comida, Limpieza, Higiene, Snacks y Mascotas.
- Las fechas de las ventas se distribuyen aproximadamente entre 2023 y 2024.
